# Programa de Inglés ChuyMath — Roadmap Global

Roadmap completo desde nivel cero hasta B1, basado en Cambridge Young Learners (YLE) + CEFR.
Cada nivel tiene su documento detallado con las 40 unidades y vocabulario específico.

## Estructura de niveles

```
A0  → Pre-Starter     (Pre-A1)         ← Primeras palabras
A1-1 → Starter         (Pre-A1 Starters) ← NIVEL ACTUAL DEL ALUMNO
A1-2 → Elementary      (A1 Movers)      ← Siguiente nivel
A2   → Pre-Intermediate (A2 Flyers)     ← Meta a mediano plazo
B1   → Intermediate    (KET for Schools) ← Meta a largo plazo
```

|-----------|-------------------|
| **Comparatives** | bigger than, more beautiful than, the biggest |

## Nivel A2: Pre-Intermediate (estimado 40 unidades)

**Edad típica:** 9-11 años | **Cambridge:** A2 Flyers | **Vocabulario:** ~1200 palabras (+ ~400 nuevas)

### Gramática nueva (sobre A1-2)
Basado en Cambridge A2 Flyers Grammar List + CEFR A2:

| Categoría | Estructuras nuevas |
|-----------|-------------------|
| **Present Perfect** | I have been to London. She has eaten lunch. |
| **Present Perfect vs Past Simple** | I've seen that movie (experience) vs I saw it yesterday (specific time) |
| **Past Continuous** | I was playing when mom called. |
| **Used to** | I used to live in Mexico. She didn't use to like vegetables. |
| **Have to / Don't have to** | You have to wear a uniform. You don't have to pay. |
| **Should / Shouldn't** | You should study more. You shouldn't eat so much candy. |
| **Might / May** | It might rain tomorrow. She may come to the party. |
| **First Conditional** | If it rains, I'll stay home. |
| **Defining relative clauses** | The boy who lives next door. The book that I read. |
| **Passive voice (simple)** | This house was built in 1990. English is spoken here. |
| **Too / Enough** | It's too cold. She's old enough to drive. |
| **So / Neither** | So do I. Neither can she. |

### Temas de vocabulario nuevos
| Categoría | Palabras clave |
|-----------|---------------|
| **Technology** | computer, phone, internet, email, website, app |
| **Travel** | passport, ticket, hotel, luggage, flight, sightseeing |
| **Shopping** | price, discount, receipt, size, try on, pay |
| **Media** | newspaper, magazine, TV show, movie, advertisement |
| **Feelings avanzado** | excited, nervous, bored, disappointed, surprised, proud |
| **Social** | invitation, celebration, tradition, culture, community |

### Bloques temáticos estimados
| Bloque | Unidades | Temas |
|--------|----------|-------|
| Vocabulario nuevo | A2-01 a A2-10 | Technology, Travel, Shopping, Media, Feelings, Social, Nature, Food extended, Professions extended, Home extended |
| Gramática nueva | A2-11 a A2-20 | Present Perfect, PP vs Past Simple, Past Continuous, Used to, Have to, Should, Might/May, First Conditional, Relative clauses, Passive voice |
| Comunicación | A2-21 a A2-30 | Writing short messages, Describing experiences, Making suggestions, Giving advice, Expressing preferences, Telling stories, Comparing past and present, Making predictions, Agreeing/Disagreeing formally, Sequencing events |
| Integración | A2-31 a A2-40 | Reviews, Long stories, Expediciones avanzadas, Assessment tipo Cambridge Flyers |

## Flujo de trabajo

1. Al terminar un nivel, hacer pesquisa detallada del siguiente
2. Crear `PROGRAMA_INGLES_{NIVEL}.md` con 32-40 unidades detalladas
3. Crear contenido JSON para cada unidad (hasta 9 tipos de juego por unidad)
4. Si el maestro asigna un tema específico, ubicarlo en el programa y priorizar
